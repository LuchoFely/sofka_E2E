# Proyecto de Automatización con Katalon Studio

## 📌 Descripción

Este proyecto tiene como objetivo automatizar pruebas funcionales para la aplicación https://www.demoblaze.com/ utilizando **Katalon Studio**. Las pruebas cubren 

• Agregar dos productos al carrito
• Visualizar el carrito
• Completar el formulario de compra
• Finalizar la compra

## 🛠️ Tecnologías utilizadas

- Katalon Studio


## 🚀 Cómo ejecutar las pruebas


Abre el proyecto en Katalon Studio.


Ejecuta los casos de prueba desde el panel de Test Suites o directamente desde los Test Cases.
